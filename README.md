# context
A port of the [jQuery](http://api.jquery.com/jQuery.proxy/) proxy functionality to NodeJS

### Installation
The easiest way to install is through the Node Package Manager ([NPM](http://npmjs.org/)):

``` js
npm install context
```

### Usage
``` js 
var context = require('context');

function helloWorld(){
	console.log(this);	
}



```
